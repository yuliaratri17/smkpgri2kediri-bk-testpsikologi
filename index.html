Tes Psikologi Internal Sekolah

Aplikasi ini dibuat agar siswa dapat mengerjakan tes menggunakan email pribadi, sementara admin dapat mengelola soal dan melihat rekap hasil. Aplikasi ini cocok untuk diunggah ke GitHub Pages sebagai satu file HTML.

Input soal teks / gambar Simpan progres per email Rekap ke spreadsheet Mode super admin & admin

Masuk sebagai siswa

Progres disimpan di perangkat ini, dan bisa dikirim ke spreadsheet jika endpoint diset di mode admin.

Petunjuk penggunaan

1. Siswa mengisi email pribadi lalu memulai tes.
2. Admin login untuk menambah, mengubah, atau menghapus soal.
3. Hasil tes tersimpan otomatis di browser, dan bisa dikirim ke Google Sheets / spreadsheet lain.
4. File ini bisa langsung dijadikan index.html dan dipublish lewat GitHub Pages.

Status aplikasi

Catatan: login admin di HTML statis cocok untuk penggunaan internal sekolah. Untuk keamanan tinggi, sebaiknya nanti dipindahkan ke backend / Firebase Auth / Google Login.
